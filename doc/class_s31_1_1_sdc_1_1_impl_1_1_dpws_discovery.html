Detailed Description

Main processor for WS-Discovery handling in S31.Core.

It combines WS-Discovery handling as well as network configuration changes and simplifies the interface to just found and lost devices (S31::Sdc::Impl::IDpwsDiscoveryObserver).

This needs a IScheduling to be able to do periodic probes in case a discovery proxy is used.

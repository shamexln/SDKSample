void S31::AsioDispatch::DefaultDispatcher::startTasks ( )

Allows task processing again.

Does not need to be called initially.

This function is needed because classes around have start()/stop() and so are (in theory) able to restart.

◆ stopTasks()

void S31::AsioDispatch::DefaultDispatcher::stopTasks ( )

Stops the task processing in the dispatcher.

Pending tasks are canceled. Contained task function may or maybe not be executed. New Tasks cannot be added (adding will succeed but without effect).

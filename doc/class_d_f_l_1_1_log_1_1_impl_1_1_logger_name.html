Detailed Description

Identity type of a logger.

This type is used to identify a specific instance of an DFL::Log::Logger. LoggerName models a hierarchy similar to a folder path.

const LoggerName abcStr{"a/b/c"};
const LoggerName& abcOp = LoggerName{"a"} / "b" / "c";
EXPECT_THAT(abcOp, Eq(abcStr));

Punctuations are also allowed in a name:

EXPECT_NO_THROW({ const LoggerName a{LoggerName{"a_b_c"} / "b_b_c" / "c_b_c"}; }); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_NO_THROW({ const LoggerName a{LoggerName{"a-b-c"} / "b-b-c" / "c-b-c"}; }); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_NO_THROW({ const LoggerName a{LoggerName{"a.b.c"} / "b.b.c" / "c.b.c"}; }); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_NO_THROW({ const LoggerName a{LoggerName{"a#b#c"} / "b#b#c" / "c#b#c"}; }); // NOLINT(cppcoreguidelines-avoid-goto)

Not all characters are valid in a name. Here are some examples of invalid ones:

EXPECT_THROW({ const LoggerName a{" "}; }, std::runtime_error); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_THROW({ const LoggerName a{"/"}; }, std::runtime_error); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_THROW({ const LoggerName a{"//"}; }, std::runtime_error); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_THROW({ const LoggerName a{"a//"}; }, std::runtime_error); // NOLINT(cppcoreguidelines-avoid-goto)
EXPECT_THROW({ const LoggerName a{"/ "}; }, std::runtime_error); // NOLINT(cppcoreguidelines-avoid-goto)

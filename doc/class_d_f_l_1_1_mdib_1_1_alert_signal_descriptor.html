Detailed Description

Defines the dynamic part of an alert signal.

An alert signal contains information about the way an alert condition (see AlertCondition) is communicated to a human. It is generated by an alert system (see AlertSystem) to indicate the presence or occurrence of an alert condition.

Indicates the acknowledgment timeout if the signal supports acknowledgment.

This value is only operative if AlertSignalDescriptor::acknowledgementSupport() returns true. The acknowledgment timeout is given as a duration or is set to boost::none if the duration is infinite.

Flag to access if a signal is latching or not.

An alert signal is latching if it outlives its triggering alert condition until it is stopped by deliberate action.

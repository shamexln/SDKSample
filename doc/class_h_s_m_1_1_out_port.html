Detailed Description

template<class T, unsigned short replFactor = 1>
class HSM::OutPort< T, replFactor >

Output port, through which state machines sends events to others.

Replication factor defines the "FAN OUT", i.e. to how many inports the outport can be connected to simultaneously.

Template Parameters
TInterface definition class for events received through this port
replFactorMax number of ports that can be connected to this port

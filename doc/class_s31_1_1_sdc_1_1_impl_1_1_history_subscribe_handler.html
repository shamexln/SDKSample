void S31::Sdc::Impl::HistorySubscribeHandler::subscriptionEnded ( const EndData &  endData)
overridevirtual

Information about the end of a subscription.

This function will only be called for situation that are not explicitly request locally. These are subscription expiration, unsubscribe and transmission errors. If endSubscription() was called to explicitly end a subscription no such call will be made. Also in case of destruction no callback will be made. To gracefully end a subscription call endSubscription with SubscriptionEndReason::SourceShuttingDown for example.

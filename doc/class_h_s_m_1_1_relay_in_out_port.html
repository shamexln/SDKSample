Detailed Description

template<class InType, class OutType, unsigned short replFactor = 1>
class HSM::RelayInOutPort< InType, OutType, replFactor >

A port which just forwards incomming events.

This is useful for connecting containers with eachother and makes it possible to organize a group of event handlers into components with less complex in-/outputs.

Template Parameters
InTypeInterface definition class for the events received by a container through this port.
OutTypeInterface definition class sent by a container through this port.
replFactorMax number of ports that can be connected to this port.

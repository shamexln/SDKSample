Detailed Description

Https POST Handler interface.

This interface provides for application specific handling of POST requests received by a S31::Sdc::Impl::HttpServer.

Handle Http Post requests.

This method is called by the http server to respond to Post requests.

Note
The message body may be compressed (which is indicated by the contentEncoding). Responsibility to decompress the body is delegated to class implementing this interface.

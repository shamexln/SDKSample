Detailed Description

Storage for LocalizedTextReference.

For usage on client side. Not thread safe.

Member Function Documentation

◆ add()

void S31::Localization::LocalizedTextReferenceStore::add ( const LocalizedTextReference &  ref)

Adds the given ref to the store.

Handles 'is more general' semantics. That is that:

  • if there is a more general reference already in the store, then nothings is added
  • More specialized references in the store will be removed

◆ isCovered()

bool S31::Localization::LocalizedTextReferenceStore::isCovered ( const LocalizedTextReference &  ref) const

Checks if the given reference is covered by the store.

Note
Special case of a store that contains all textwidths (for one ref, version, and number of lines): Then this function will return false for the according reference with no textwidth set.
Returns
returns true if ref or a more general reference is contained in the store; false otherwise

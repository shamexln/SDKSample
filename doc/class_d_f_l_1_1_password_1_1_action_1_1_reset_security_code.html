Detailed Description

Clears the according securitycode and failed login attempts counter in the persistence.

::DFL::Password::Action::ResetSecurityCode resetSecurityCode{group};
if (!resetSecurityCode.reset())
{
FAIL() << "reset security code failed";
return;
}
if (!resetSecurityCode.isReset())
{
FAIL() << "reset security code failed";
return;
}

bool DFL::Password::Action::ResetSecurityCode::isReset ( ) const

Returns true if securitycode is reset otherwise false.

Securitycodes with an empty saltedhash are considered to be reset.

EXPECT_TRUE(env.m_changeAction.save(env.m_goodSecurityCode));
EXPECT_FALSE(env.m_resetAction.isReset());

◆ reset()

bool DFL::Password::Action::ResetSecurityCode::reset ( )

Returns true if reset was successful otherwise false.

On success the hashed securitycode is empty and the counter for the failed login attempts is 0 in the persistence.

EXPECT_TRUE(env.m_resetAction.reset());
EXPECT_TRUE(env.m_resetAction.isReset());

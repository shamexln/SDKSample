Detailed Description

Observer to receive information about remote devices.

The information that is received on DiscoveryObserver::foundDevice() can be used to ultimately connect to a remote device by using S31::Sdc::RemoteDevicesProvider.

When using byeFromDevice() it should be considered that this notification is based an a unreliable/unencrypted multicast message.

Note
The information that is conveyed on DiscoveryObserver::foundDevice() is announced by the discovery provider without any further communication to the remote device.

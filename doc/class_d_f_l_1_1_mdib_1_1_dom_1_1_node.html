Detailed Description

Models an xml element or text value which is part of an extension to BICEPS.

Each node identifies its content as either an DFL::Mdib::Dom::Element or text value. Nodes are ordered in depth first traversal order (i.e. a parent node is immediately followed by its first child node).

A depth field distinguishes a parent node from its children. Adjacent nodes with equal depth are siblings.

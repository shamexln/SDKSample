Set the presence lists in the affected AlertSystems.

The provided change set may contain alert conditions with changed presence (AlertConditionState::presence()). The alert systems containing these alert conditions hold lists of present alert conditions (AlertSystemState::presentPhysiologicalAlarmConditions(), AlertSystemState::presentTechnicalAlarmConditions()). This function sets these lists so that they reflect the new Mdib.

This function should be used beforehand of Mdib write operations to local Mdib: LocalMdibAccess::writeDescription(), LocalMdibAccess::writeAlertStates()

Parameters
[in]mdibThe old Mdib
[in,out]changeSetChangeset to be merged into Mdib.
[in]modusShould the old MDIB considered consistent, i.e. are it's alert system presence lists right or should all be build from scratch

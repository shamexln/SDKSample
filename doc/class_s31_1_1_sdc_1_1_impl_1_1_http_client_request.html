Detailed Description

Provide parameters to an http client request operation.

This class provides the parameters for composing an http request, including a callback function to execute upon completion. Parameters include the http method, target endpoint address, and optional message body and header fields appropriate for the method. It also provides some customization of behavior on a per-request basis, such has additional credentials checks.

Note
This class is not intended for direct use by application code, but to be constructed using the HttpClientRequestBuilder class, and provided to a HttpClient.

Parameters
methodThe HttpMethod for the request.
serverAddressThe authority component from a URI (minus user info) which the request will be sent to. It is formatted as host:port
targetPathThe path and query components from a URI which is transmitted as the target path of the request. It must be an absolute path starting with '/'
acceptEncodingThe HttpAcceptEncoding header is optional and will be ignored if an empty string is provided.
requestBodyThe body for the request is optional and will be ignored if an empty string is provided.
contentTypeThe HttpContentType must be provided if a body value is provided, but will be ignored if the message body is empty.
contentEncodingThe HttpContentEncoding must be provided if the body value has additional encoding (e.g. compression) but will be ignored if the message body is empty.
retryAllowedIndicate whether or not automatic retry of the request is allowed in case of connection loss.
verifyPreconditionsFuncIs an optional function (i.e. may be empty) additional per-request verification of the HttpSessionContext (e.g. additional credentials verification) which is called before the request is sent.
onCompleteCallbacksFunctions which are called when the response to a request is received, or a final error is detected. It provides the HttpResult to the consumer of http requested data.
Note
It is recommended to use a HttpClientRequestBuilder to configure the request, rather than calling this constructor directly.

Detailed Description

Mapper between BicepsXml and History interfaces for MDIB changes.

The observer interface from the BicepsXml component is different (more complex) due to the fact that it uses the subdivided observer interface and has shared_ptr in its callback functions. The History service interface is by design without shared_ptr to enforce implementations to copy the data into an own format and not storing the shared_ptr.

Detailed Description

A pool of gSOAP soap contexts.

Use this pool for all required soap contexts used for communication in dSOAP. Additional functionality (gSOAP plug-ins) will be used for WS-Addressing for example. Later on it may be possible to add HTTP keep-alive for example.

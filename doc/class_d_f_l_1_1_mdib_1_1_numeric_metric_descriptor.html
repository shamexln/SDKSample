const boost::optional< Duration > & DFL::Mdib::NumericMetricDescriptor::averagingPeriod ( ) const
inline

Timespan from where the measured values are used to determine the metric's value by averaging with some algorithm.

The averaging period defined in the descriptor might be not the currently active averaging period. The active averaging period is part of NumericMetricState.

Returns
Timespan used to average metric values.

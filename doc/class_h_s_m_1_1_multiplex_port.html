Detailed Description

template<class T, unsigned short replFactorOut = HSM::DYNAMIC_SIZE, unsigned short replFactorIn = 1>
class HSM::MultiplexPort< T, replFactorOut, replFactorIn >

MultiPlexPorts can be used to broadcast events received through the In part to any port connected to the Out part.

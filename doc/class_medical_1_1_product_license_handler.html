Detailed Description

Interface for installing and deinstalling product licenses.

As the installation is processed asynchronically, this interface also provides information about the installation progress and result.

◆ processProductLicenses()

static bool Medical::ProductLicenseHandler::processProductLicenses ( const ProductLicenseContainer &  licenses)
static

Triggers installation or deinstallation of product licenses, depending on the InstallDirection flag in a license.

For the current implementation, the ProductLicenseContainer parameter should contain only one license! Only one license will be installed or deinstalled.

Parameters
licensesInstallation or deinstallation is triggered for these licenses
Returns
true if installation or deinstallation process could be started, other wise false

const XPathExpressions & DFL::Mdib::AbstractSetStateOperationDescriptor::modifiableData ( ) const
inline

Describes a list of members the underlying operation modifies on invocation by means of XPath expressions.

Semantics:

  • If the modifiable data list is empty, then all members of the target state shall be modifiable.
  • The root element of the XPath expressions is the state referenced by AbstractOperationDescriptor::operationTarget

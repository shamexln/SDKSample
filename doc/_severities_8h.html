Detailed Description

Definition of different logging severities.

Trace, success, info, warning and error are the different severities. In that order these severities make an ascending sequence. This is used for filtering messages.

DFL_LOG_SUCCESS_WITH(logger, "This message gets logged");
// Trace, Success and Info shall no longer be logged
logger.severity(Warning);
DFL_LOG_SUCCESS_WITH(logger, "This message is not processed");

Detailed Description

UDI fragments as defined by the FDA.

An example UDI in human readable form (GS1 barcode) is "(01)51022222233336(11)141231(17)150707(10)A213B1(21)1234". The "(01)" application identifier identifies the DeviceIdentifier (DI) of the UDI. Thus in the example the DI is "51022222233336".

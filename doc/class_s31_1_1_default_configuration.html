static unsigned int S31::DefaultConfiguration::maxIdleIncomingConnections ( )
inlinestatic

Maximum number of idle keep-alive incoming connections.

One a device it covers requests such as GetMDIB or WS-Eventing subscribe. In a client it is the max numbers of notifications connections.

◆ maxIdleOutgoingConnections()

inlinestatic

Maximum number of idle keep-alive outgoing connections.

It covers connections from a device to the client (event sink) as well as from a client to a device (e.g., GetMDIB calls)

◆ CodePoint

using DFL::Locale::Utf8Constants::CodePoint = unsigned int

Type for code points for UTF-8 encoding.

The code point in UTF-8 represents the unicode code point. It is encoded with multibyte encoding (1 to 4 byte). Code points have up to 21 bit, but not all values are allowed.

UTF-8 encoding of replacement character (U+FFFD).

The replacement character is special. It is used in UTF-8 decoding for invalid byte sequences. Hint: Do not use this character in identifier. http://www.fileformat.info/info/unicode/char/fffd/

Detailed Description

Container for UTF-8 encoded string.

Container with a valid UTF-8 encoded string. All instances contain valid UTF-8 data only.

DFL::Locale::Utf8::Utf8 ( const std::string &  utf8String)
explicit

Conversion uses REPLACEMENT CHARACTER (U+FFFD) for invalid byte sequences.

This implicit conversion from std::string to Utf8 converts invalid byte sequences to the REPLACEMENT CHARACTER (U+FFFD). Thus no error occur and valid UTF-8 encoding is assured.

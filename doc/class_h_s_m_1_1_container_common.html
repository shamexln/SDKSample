Detailed Description

Base class for containers which aggregate multiple statemachines.

Designed according to the composite pattern, which means that containers can aggregate (sub-)-containers and statemachines. Enables structuring of large subsystems and dynamic subsystems. For dynamic subsystems which are constructed after the owning active object is already initialized, the init-method has to be called after constructing the container and its members. The constructor registers all contained statemachines and the statemachines of any subcontainers at the active object, this needs not to be done.

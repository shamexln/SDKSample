Detailed Description

Helper class to simplify building a tree of Nodes.

This class facilitates building a sequence of DFL::Mdib::Dom::Node as a tree in depth first traversal order, where the DFL::Mdib::Dom::NodeDepth of a DFL::Mdib::Dom::Node indicates the parent-child or sibling relationship between adjacent Nodes.

The TreeBuilder is to simplify handling the depth of the next Node to be added to the tree.

Increase the depth of the DFL::Mdib::Dom::Node tree to add the children of an DFL::Mdib::Dom::Element.

Elements and texts added after the method is called will be saved as children of the node added immediately preceding the call. It is required that for each call there is a preceding Element already added to the tree.

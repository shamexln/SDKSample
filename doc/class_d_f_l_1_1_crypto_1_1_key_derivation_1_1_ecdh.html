Detailed Description

Provides shared secret creation based on ECDH.

For details on Elliptic Curve Diffie Hellman (ECDH) see https://en.wikipedia.org/wiki/Elliptic-curve_Diffie%E2%80%93Hellman.

Warning
The shared secret should not be used directly as a symmetric key, but it can be used as entropy for a key derivation function.
const Ecdh ecdh{};
const auto aliceSharedSecret{ecdh(alicePrivateKey, bobPublicKey)};
const auto bobSharedSecret{ecdh(bobPrivateKey, alicePublicKey)};
EXPECT_EQ(aliceSharedSecret, bobSharedSecret);

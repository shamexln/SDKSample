Detailed Description

Context for comparing the semantics of DFL::Mdib::CodedValue.

Two CodedValues are nominally equal in case that

  • the coding system, code and coding system version matches or
  • a translation in one CodedValue is equal to the other CodedValue, or one of its translations.

However, in general, assumptions cannot be made about private codes in ISO/IEEE 11073, or unknown coding systems. Only public ISO/IEEE 11073 will compare equal, unless it can be inferred (via search for DFL::Mdib::ProductionSpecification) that ISO/IEEE 11073 private codes have Draeger private code semantics.

Note
Coding system name and description (both LocalizedText) are not used for comparison.

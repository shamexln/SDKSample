Detailed Description

A uniformly-distributed random bit generator (URBG) that uses OpenSSL RAND_bytes as random source.

Implements the concept UniformRandomBitGenerator (see https://en.cppreference.com/w/cpp/named_req/UniformRandomBitGenerator).

std::uniform_int_distribution<int> dice{1, 6};
OpenSslUrng roll{};
std::vector<int> rolls(10, 0);
std::generate(rolls.begin(), rolls.end(), [&]() mutable {
return dice(roll);
});
ASSERT_TRUE(std::all_of(rolls.cbegin(), rolls.cend(), [](int a) {
return a >= 1 && a <= 6;
}));

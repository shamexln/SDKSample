Detailed Description

Offers a fluent interface for building message objects.

This is especially useful when writing your own logging filter. In most cases you just want to update only parts of the given message and leave the rest as is.

const Text newTxt{"new formatted text"};
const Message m{MessageBuilder{m_msg}.text(newTxt)};
EXPECT_NE(m, m_msg);
EXPECT_EQ(MessageComparator{m_msg}.text(m.text()), m);

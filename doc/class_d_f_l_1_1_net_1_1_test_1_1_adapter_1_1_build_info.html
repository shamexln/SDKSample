Detailed Description

Builder for creating a fake adapter info object.

Objects are created using a fluent interface. Intended to be used for construction of MockInfos objects.

const MockInfos mock{
BuildInfo{}
.index(1)
.name("adapter_a")
.isLoopback(false)
.isMulticast(true)
.isConnected(true)
.addressesV4(
AddressesV4{AddressV4{"1.2.3.4"}, AddressV4{"11.22.33.44"}})
.macAddress(MacAddress{{1, 2, 3, 44, 55, 66}}),
BuildInfo{}
.index(22)
.name("adapter_b")
.isLoopback(true)
.isMulticast(false)
.isConnected(false)
.addressesV4(AddressesV4{AddressV4{"5.6.7.8"}})
.macAddress(MacAddress{{5, 6, 7, 88, 99, 111}})};

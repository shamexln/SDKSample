Detailed Description

MetadataVersion class for atomic use.

This class provides a wrapper for an atomic (32 bit) unsigned int while exposing the Strong Typedef S31::Sdc::MetadataVersion to the outside. Additionally, it restricts operations on the underlying data to those that are sensible for metadata versions, e.g. it does not allow decrementing.

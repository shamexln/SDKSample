Detailed Description

Indicate that the waveform data in the element is normalized in some way.

This extension is intended to be attached to pm:RealTimeSampleArrayDescriptor elements. It indicates that the waveform data in the element is normalized in some way.

Note
An SDC SERVICE PROVIDER shall set @ext:MustUnderstand = "true".

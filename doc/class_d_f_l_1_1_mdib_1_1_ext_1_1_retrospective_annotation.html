Detailed Description

Annotations to samples of previous waveform frames.

RetrospectiveAnnotation allows for an SDC Service Provider to apply annotations to samples of previous waveform frames.

SampleIndex DFL::Mdib::Ext::RetrospectiveAnnotation::sampleIndex ( ) const
noexcept

Index number that points to a sample in the past.

Index number that points to a sample in the past to which the annotations in Annotation are to be applied to retrospectively. The referenced sample can be calculated by the following formula: ReferencedSample = OffsetSample + SampleIndex, where OffsetSample points to the first sample in the @Samples list of the pm:RealTimeSampleArrayMetricState element in which the referenced annotation is included. ReferencedSample virtually references a sample from the past, namely at the time T when the sample was measured, with T = pm:RealTimeSampleArrayMetricState/@DeterminationTime - |SampleIndex| * pm:RealTimeSampleArrayMetricDescriptor/@SamplePeriod.

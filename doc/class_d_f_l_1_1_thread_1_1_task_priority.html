Detailed Description

Provides an enumeration of task priorities based on task response time requirements.

For fast periodical tasks, the rate can correspond to the period. For episodic respective event driven tasks, the rate should correspond to the response time.

Warning
But for every task holds, that the chosen rate should never be exceeded.

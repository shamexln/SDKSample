Detailed Description

A decimal number that is truncated at a certain fractional digit.

Truncation rules with FractionalDigits:

  • If the fractionalDigits value is absent, no truncation is performed
  • If the fractionalDigits value is present, it specifies the number of digits to the right of the decimal point
  • If the actual fractional digits of the decimal are less than the fractional digits to truncate, the rendered decimal is padded with zeros to the right to meet the number of digits specified by fractionalDigits

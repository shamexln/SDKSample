Detailed Description

Combines kind and instance of a component to identify a unique instance of a component.

Instances of this class should not be created via a constructor. Instead compiletime variants are provided.

static Kind::NativeType const KindValue = 0x17;
static Instance::NativeType const InstanceValue = 0x2;
using KindName = Static::string<'K', 'i', 'n', 'd'>;
using KindType = StaticKind<KindValue, KindName>;
using InstanceType = StaticInstance<InstanceValue>;
using KeyType = StaticKey<KindType, InstanceType>;

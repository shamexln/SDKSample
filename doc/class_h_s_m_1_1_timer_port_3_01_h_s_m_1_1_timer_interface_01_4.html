Detailed Description

Implementation of a timer input port sepecialized with the HSM::TimerInterface.

This timer port will deliver HSM_TIMEOUT and as periodical tick HSM_TICK to the owning state machine.

Member Function Documentation

◆ startPeriodicalTick()

void HSM::TimerPort< HSM::TimerInterface >::startPeriodicalTick ( unsigned int  divider = 1)
inline

When instantiated on a periodical task, this method starts sending events every n-th run of the task.

This method only works on periodical threads! Contrary to startTimerPeriodically the period is strictly coupled to the period of the underlying thread. Delivers the event HSM_TICK unless a user supplied interface definition is used to instantiate the timer.

Starts a periodical timer with a delay and period given by the sum of the parameters.

The period is strictly with respect to the system time (as provided by the TimerImpl). If no user supplied interface definition is used the event HSM_TIMEOUT is delivered to the owning state machine.

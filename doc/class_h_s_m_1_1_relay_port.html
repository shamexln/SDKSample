Detailed Description

template<class InType, class OutType, unsigned short replFactor>
class HSM::RelayPort< InType, OutType, replFactor >

Base class for ports that forward events to another port.

Can also be used for ports that convert events from other sources to HSM events.

Template Parameters
InTypeInterface definition class for the events received through this port.
OutTypeInterface definition class sent through this port.
replFactorMax number of ports that can be connected to this port.

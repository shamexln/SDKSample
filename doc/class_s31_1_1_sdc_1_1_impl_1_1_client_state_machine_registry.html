Detailed Description

Hosts client state machines and manages their life cycle.

Moreover, provides a function to retrieve existing remote devices.

Constructor & Destructor Documentation

◆ ~ClientStateMachineRegistry()

S31::Sdc::Impl::ClientStateMachineRegistry::~ClientStateMachineRegistry ( )
overridenoexcept

Sends a S31::Sdc::Errors::Shutdown offline error for all connected remote devices without disconnecting beforehand.

If you want to perform a graceful shutdown, invoke ClientStateMachineRegistry::disconnect() in advance.

Starts connection process if none is running.

  • If at the same time a disconnect is running, the connect is postponed until the disconnect is finished

    • priorityGroup configures a connection delay, allowing clients to connect to a device in priority order

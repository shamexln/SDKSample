Detailed Description

Used by an SDC SERVICE PROVIDER to communicate texts in which placeholder tokens which are replaced by specific values.

Allows for fewer translations when different texts are needed for a labeled parameter. Also allows for richer alarm text to be shown to the user ("HR > 80 vs "HR HIGH"")

Example usage in code:

// Device side usage: create the PlaceholderText instance with free function
// DFL::Mdib::Ext::ParticipantModelExtensions::toExtension.
// Add it to any MDIB class that contains LocalizedTexts that need placeholders.
// Here AlertConditionDescriptor.type() is a CodedValue that contains
// Localizedtexts. This is a valid use case that may require Placeholders
DFL::Mdib::AlertConditionDescriptorBuilder valueBuilder;
// Text or Text Reference will contain Placeholder Tokens.
// Placeholder tokens are markers that are enclosed by dollar characters ($)
// Usually Texts or TextReferences, here both examples are shown.
const auto localizedTexts = DFL::Mdib::LocalizedTexts{ DFL::Mdib::LocalizedText{DFL::Locale::Utf8("$LABEL$ > $LIMIT$")},
DFL::Mdib::LocalizedText{DFL::Mdib::LocalizedTextRef{DFL::Locale::Utf8("1234")}, boost::none} };
// Placeholders contain a Token and either Text or a Number
// FractionalDigits specifies how many digits after the decimal point the Decimal value.
// Here, Decimal should be truncated to 15.5 since FractionalDigits == 1
const auto placeholders = DFL::Mdib::Ext::Placeholders{
DFL::Mdib::Ext::Placeholder{DFL::Mdib::Ext::PlaceholderToken{"LABEL"},
"Trect"},
DFL::Mdib::Ext::Placeholder{DFL::Mdib::Ext::PlaceholderToken{"LIMIT"},
DFL::Mdib::Ext::DecimalWithFractionalDigits{DFL::Mdib::Decimal{15},
DFL::Mdib::Ext::FractionalDigits{1}}} };
// Set the extensions in the PlaceholderText
DFL::Mdib::Ext::ParticipantModelExtensions::setExtension(
valueBuilder.extensions(),
PlaceholderText{ localizedTexts,
placeholders });
auto value = valueBuilder.build();
// Client side usage: use the free function
// DFL::Mdib::Ext::ParticipantModelExtensions::get<PlaceholderTextUsage>() to
// retrieve the PlaceholderTextUsage from a any extentable MDIB element.
const auto placeholderText =
DFL::Mdib::Ext::ParticipantModelExtensions::get<PlaceholderText>(value);
if (placeholderText)
{
EXPECT_THAT(placeholderText->texts(), Eq(localizedTexts));
EXPECT_THAT(placeholderText->placeholders(), Eq(placeholders));
}

Localized texts or text references containing placeholder tokens.

Placeholder tokens are markers that are enclosed by dollar characters ($). Example: $LIMIT$. Placeholder Tokens are there to be replaced with placeholder values. Multiple texts allowed for multiple languages.

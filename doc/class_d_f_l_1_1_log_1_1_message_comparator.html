Detailed Description

Stores given message attributes and compares each given message against those attributes.

This is useful if you are only interested in some particular attributes of a message:

using namespace ::DFL::Log;
::DFL::Log::Test::MockLogger const mock(myLogger);
LOG_WITH(myLogger, Info, "text1");
LOG_WITH(myLogger, Info, "text2");
LOG_WITH(myLogger, Warning, "text1");
LOG_WITH(myLogger, Warning, "text2");
LOG_WITH(myLogger, Error, "text1");
LOG_WITH(myLogger, Error, "text2");
ASSERT_TRUE(mock.size() == 6);
ASSERT_TRUE(3 == std::count(mock.cbegin(),
mock.cend(),
MessageComparator{}.text(Text{"text1"})));
ASSERT_TRUE(2 == std::count(mock.cbegin(),
mock.cend(),
MessageComparator{}.severity(Warning)));
ASSERT_TRUE(1 == std::count(mock.cbegin(),
mock.cend(),
MessageComparator{}.severity(Warning).text(
Text{"text1"})));

On the other hand if you want to check that all attributes have been left unchanged and only some have been updated:

const Text newTxt{"new formatted text"};
const Message m{MessageBuilder{m_msg}.text(newTxt)};
EXPECT_NE(m, m_msg);
EXPECT_EQ(MessageComparator{m_msg}.text(m.text()), m);

A default constructed instance always returns true no matter what message it is compared to.

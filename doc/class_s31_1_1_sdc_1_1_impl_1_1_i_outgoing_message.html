This function is called from the WS-Eventing plug-in for each subscriber (endpoint).

The customer implementation must use the gSOAP generated proxy class to send the message to the given endpoint/interface. This function cannot be const because it's implementations calls a non-const (gSOAP) interface.

Parameters
soapHandleThe soap context is already set up
endpointTo this endpoint the event must be delivered (with help of the gSOAP generated proxy classes for the specific service)
Returns
the gSOAP error code returned for delivering the event to the subscriber

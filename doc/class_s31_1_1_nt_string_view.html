Detailed Description

Null terminated string view.

This class is similar to std::string_view, except that is guarantees to be a null terminated string view. Very often there are some constants in S31 that are null terminated strings. In some places std::string is used with heap allocation and in other places const char* is used. This string view class allows to use constexpr and has string_view like equality operator for example for easy usage of the constant.

Detailed Description

template<class PseudorandomFunc>
class DFL::Crypto::KeyDerivation::Pbkdf2< PseudorandomFunc >

Provides key derivation based on PBKDF2.

For details on Password-Based Key Derivation Function 2 (PBKDF2), see https://en.wikipedia.org/wiki/PBKDF2

The number of iterations can be specified to adjust the security and the performance of this algorithm.

◆ timeSinceEpoch

TimeSinceEpoch DFL::Chrono::Registry::SystemClock::timeSinceEpoch
static

Assigning a hook to this variable will redirect the call of DFL::Chrono::SystemClock::now.

The function has to return the time which passed since the 1st jan 1970 in microseconds since this is the epoch of this clock After assignment any further calls to DFL::Chrono::SystemClock::now are passed to the assigned function.

◆ toLocal

ToLocal DFL::Chrono::Registry::SystemClock::toLocal
static

Assigning a hook to this variable will redirect the call of DFL::Chrono::Convert::toLocalDateTime.

The function has to convert the given time since epoch into a local date time object. After assignment any further calls to DFL::Chrono::Convert::toLocalDateTime are passed to the assigned function.

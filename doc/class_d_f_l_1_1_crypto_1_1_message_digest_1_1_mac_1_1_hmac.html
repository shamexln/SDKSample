Detailed Description

template<class Hash>
class DFL::Crypto::MessageDigest::Mac::Hmac< Hash >

Provides message authentication codes for messages using HMAC.

Template Parameters
Hashhash function like Sha256.

For details on hash-based message authentication code (HMAC) see https://en.wikipedia.org/wiki/HMAC

Hmac<Sha256> hmac{hmacKey};
for (int i = 0; i < 2; ++i)
{
hmac.init();
for (const auto& b : messageBytes)
{
hmac.update(BytesView{&b, 1});
}
hmac.final();
ASSERT_TRUE(verify(macBytes, hmac.mac()));
// first call of mac() consumed already all bytes
ASSERT_TRUE(hmac.mac().empty());
}

Detailed Description

Interface to register for MDIB changes of MdibAccess subject.

Attention
This interface is in accordance to the message model of SDC. Please do not use it if you are not confident with SDC internals!

Right now for a writeDescription() call a DescriptionModificationReport callback as well as optional callbacks for EpisodicAlertReport, EpisodicComponentReport, ... are created. Later on, when that behaviour is changed for SDC, only the DescriptionModificationReport would be required and called on the observer.

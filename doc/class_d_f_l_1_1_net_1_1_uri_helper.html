static Uri::string_type DFL::Net::UriHelper::decode ( const Uri::string_type &  percentEncoded)
static

Decodes a given percent-encoded string.

Never decode a string twice since %25 will decode to %. Decoding twice will lead to whatever comes after % to be interpreted as percent encoded. As opposed to encoding, there is no scheme-specific decoding method. This is because the percent character, by design, is always part of the reserved character set for percent-encoded strings. Therefore, the decoding procedure is the same independent of encoding. Here are examples of percent-encoded strings and their decoded form:

{
EncodedString("%7C%5E%7E%60%3C%3E%22%7B%7D"), // characters in encoded form can be decoded
RawString("|^~`<>\"{}")
},
{
EncodedString("|^~`<>\"{}"), // the same characters in non encoded form
RawString("|^~`<>\"{}")
},
{
EncodedString("%FF%80"), // 8 bit-encodings shall work
RawString("\xFF\x80")
},
{
EncodedString("%20()%40%2C"),
RawString(" ()@,")
},
{
EncodedString("%20%28%29%40%2C%3B%3A%5C%2F%5B%5D%3F%3D%23%21%24%26%27%2A%2B%25%7C%5E%7E%60%3C%3E%22%7B%7D"), // all characters from URL reserved set
RawString(" ()@,;:\\/[]?=#!$&'*+%|^~`<>\"{}")
},
{
EncodedString("%25"),
RawString("%")
},
{
EncodedString("%44%65coding%20%61%6E%79%20enco%64%65d char%20%77%6F%72%6B%73%21%21!1%31%65%6C%66"),
RawString("Decoding any encoded char works!!!11elf")
},
Returns
Decoded String.

Percent encodes a given string using the reserved character set provided.

Apply this to parts of an URI only. Otherwise, delimiters may be encoded (usually, they are part of the reserved character set). The minimal reserved character set for URIs is defined by DFL::Net::UriHelper::uriReservedCharacterSet and will be used as the default. Here are examples of URIs and their percent-encoded form:

{
EncodedString("%20%28%29%40"),
RawString(" ()@")
},
{
EncodedString("%2C%3B%3A%5C"),
RawString(",;:\\")
},
{
EncodedString("%2F%5B%5D%3F"),
RawString("/[]?")
},
{
EncodedString("%3D%23%21%24"),
RawString("=#!$")
},
{
EncodedString("%26%27%2A%2B"),
RawString("&'*+")
},
{
EncodedString("%25"),
RawString("%")
},
{
EncodedString("|^~`<>\"{}"), // additional reserved characters for URL are not part of general URI's reserved character set
RawString("|^~`<>\"{}")
},
{
EncodedString("a%20b%28cde%29%40e%3Af%258"),
RawString("a b(cde)@e:f%8")
},

Other schemes may define more characters to be reserved, e.g. DFL::Net::UriHelper::urlReservedCharacterSet . Here are examples of URLs and their percent-encoded form:

{
EncodedString("%20%28%29%40"),
RawString(" ()@")
},
{
EncodedString("%2C%3B%3A%5C"),
RawString(",;:\\")
},
{
EncodedString("%2F%5B%5D%3F"),
RawString("/[]?")
},
{
EncodedString("%3D%23%21%24"),
RawString("=#!$")
},
{
EncodedString("%26%27%2A%2B"),
RawString("&'*+")
},
{
EncodedString("%25"),
RawString("%")
},
{
EncodedString("%7C%5E%7E%60%3C"), // additional reserved characters for URL
RawString("|^~`<")
},
{
EncodedString("%3E%22%7B%7D"), // additional reserved characters for URL
RawString(">\"{}")
},
{
EncodedString("a%20b%28cde%29%40e%3Af%258g%2Bh%7Ei%7Bgh123%7D"),
RawString("a b(cde)@e:f%8g+h~i{gh123}")
},
{
EncodedString("%FF"),
RawString("\xFF")
},

Note that encoding the same string twice results in unexpected behavior.

Returns
Percent-encoded string according to reserved character set.

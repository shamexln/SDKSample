Detailed Description

Type for modeling extensions of any kind.

Extensions enhance objects with any kind of information which cannot be otherwise represented by BICEPS. The ExtensionType class provides access to the qualified name of an extension which includes the namespace where the extension is defined, as well as whether or not the MustUnderstand is provided and true.

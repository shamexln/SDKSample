Detailed Description

template<class InType, class OutType, unsigned short replFactor = 1>
class HSM::CallbackInOutPort< InType, OutType, replFactor >

Aggregates a CallbackInPort and an Outport (optionally with a replication factor greater than 1) to allow easy reply of messages to their sender.

Template Parameters
InTypeInterface definition class for the events received through this port
OutTypeInterface definition class sent through this port
replFactorMax number of ports that can be connected to this port

Detailed Description

Holds the certificate chain containing leaf and intermediate certificates.

From https://knowledge.digicert.com/solution/SO16297.html

A certificate chain is an ordered list of certificates, containing an SSL/TLS Certificate and Certificate Authority (CA) Certificates, that enable the receiver to verify that the sender and all CA's are trustworthy.

The chain or path begins with the SSL/TLS certificate, and each certificate in the chain is signed by the entity identified by the next certificate in the chain.

Note that a certificate chain does not contain the root certificate (supposed to be known to the peer/verifier) and that the top most intermediate certificate is expected to be signed by a known root.

DFL::Crypto::CertificateChain::CertificateChain ( const Certificates &  certs)
explicit

Creates a chain of certificates from certs.

Root CAs are removed and at most one leaf certificate is being sorted in front of the intermediate ones.

If there is more than one leaf certificate then an exception is thrown.

const auto leafCert{Certificate{m_leafCertAsPem}};
const auto intermediateCa{Certificate{m_intermediateCaAsPem}};
const auto rootCa{Certificate{m_rootCaAsPem}};
{
const auto chain{CertificateChain{Certificates{leafCert}}};
ASSERT_FALSE(chain.empty());
ASSERT_EQ(1u, chain.size());
ASSERT_EQ(leafCert, chain.leafCert());
}
{
const auto twoLeafs{Certificates{leafCert, leafCert}};
// NOLINTNEXTLINE(cppcoreguidelines-avoid-goto)
EXPECT_THROW(CertificateChain{twoLeafs}, std::runtime_error);
}
{
const auto chain{CertificateChain{Certificates{intermediateCa}}};
ASSERT_FALSE(chain.empty());
ASSERT_EQ(1u, chain.size());
// NOLINTNEXTLINE(cppcoreguidelines-avoid-goto)
ASSERT_THROW(static_cast<void>(chain.leafCert()), std::runtime_error);
ASSERT_EQ(intermediateCa, *chain.cbegin(leafInclusive));
}
{
const auto chain{
CertificateChain{Certificates{leafCert, intermediateCa}}};
ASSERT_FALSE(chain.empty());
ASSERT_EQ(2u, chain.size());
ASSERT_EQ(leafCert, chain.leafCert());
auto it{chain.cbegin(leafInclusive)};
ASSERT_EQ(leafCert, *it);
++it;
ASSERT_EQ(intermediateCa, *it);
}
{
const auto li{CertificateChain{Certificates{leafCert, intermediateCa}}};
const auto il{CertificateChain{Certificates{intermediateCa, leafCert}}};
const auto ril{
CertificateChain{Certificates{rootCa, intermediateCa, leafCert}}};
const auto ilr{
CertificateChain{Certificates{intermediateCa, leafCert, rootCa}}};
EXPECT_EQ(li, il);
EXPECT_EQ(li, ril);
EXPECT_EQ(li, ilr);
}

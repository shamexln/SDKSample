Detailed Description

Easy creation of copyable/noncopyable types where you do not need to know if the old declare-stuff-private approach has to be used or if default/delete functions are available.

Warning
The macros open public/private sections. It is best to use these macros at the top of the class declaration and then open the appropriate sections.
class NotCopyable
{
DFL_NOT_COPYABLE(NotCopyable)
public:
// ...
NotCopyable() = default;
private:
// ...
};
DFL_NOT_COPYABLE
#define DFL_NOT_COPYABLE(Type)
Type is neither copy constructable nor copy assignable.

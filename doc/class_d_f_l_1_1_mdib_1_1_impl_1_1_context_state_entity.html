Detailed Description

template<class T_Entity, class T_DescrBuilder, class T_StateBuilder, class T_Parent>
class DFL::Mdib::Impl::ContextStateEntity< T_Entity, T_DescrBuilder, T_StateBuilder, T_Parent >

Container for one context state.

The container has a

  • handle for unique identification
  • state version that identifies the enclosed states's version
  • pointer to a state data container
Template Parameters

T_DescrBuilderDescriptor builder that belongs to the corresponding ContextDescriptorEntity.

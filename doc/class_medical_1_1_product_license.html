Detailed Description

Represents a product license.

Contains a unique identifier and information about install direction, validity, binding and version. Structure of this class may be extended in future, this is indicated by the version flag.

Constructor.

Parameters
[in]instancethe instance number
[in]componentKindthe component kind
[in]licenseNumberthe license number
[in]validitythe license validity. Should be ValidityTimer for Timer licenses.
[in]bindingthe license binding. Should be BindingNodeLocked for Timer licenses.
[in]counterthe duration of the license in days. Used for Timer licenses only. Otherwise set to 0.
[in]dateDaysthe current date of license usage in days resolution. Used for Timer licenses only. Otherwise set to 0. If S16 framework is used, use static_cast<uint16_t>(Core::SystemClock::actualTime().getTotalDays()) to get the days of the current date. Note: Due to the storage type, the date is limited to 2000-01-01 .. 2179-06-06. Note: Because the resolution is only days, this could lead to an effective interval of 1 day less (rounding down). It is recommended to just increment the paramater by 1.

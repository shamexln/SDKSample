Detailed Description

Infers patient and location context states to ensembles contexts automatically.

When there is a EnsembleContextDescriptor of type MDC_IDT_ENSEMBLE_LOCATION_INFERRED or MDC_IDT_ENSEMBLE_PATIENT_INFERRED in the MDIB, this helper will create/update ensemble context states for them according to location and patient context states.

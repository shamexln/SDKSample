virtual void S31::AsioDispatch::ExecutorObserver::onDispatcherStopping ( )
pure virtual

Notification that the dispatcher is stopping.

An ExecutorProvider will notify when its stop method is called. Observers must cancel outstanding asynchronous waiting operations to allow the system to shutdown cleanly.

Implemented in S31::AsioDispatch::Impl::Tasks, and S31::AsioDispatch::Impl::OrderedTasks.

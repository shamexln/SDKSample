Detailed Description

template<class T, unsigned short replFactor = 1>
class HSM::RelayOutPort< T, replFactor >

Convenience class to be used instead of HSM::RelayInOutPort, that allows simpler connections without using the Reversed/NotReversed members.

Template Parameters
TInterface definition class for the events sent by a container through this port.
replFactorMax number of ports that can be connected to this port.

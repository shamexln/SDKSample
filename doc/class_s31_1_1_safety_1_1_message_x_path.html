Detailed Description

XPath expression with the message SOAP12:Body element as the root.

MessageXPath can be created from a enumeration or plain DFL::Locale::Utf8.

◆ predefinedExpression()

boost::optional<MessageXPathExpression> S31::Safety::MessageXPath::predefinedExpression ( ) const

Returns the predefined (enum) expression if available.

Since multiple XPath expressions can lead to the same element, internally a normalization might be required.

Example: /msg:Activate/msg:Argument[1]/msg:ArgValue/text() points to the same value as /msg:Activate[1]/msg:Argument[1]/msg:ArgValue/text().

Returns
The predefined MessageXPathExpression if resolvable, otherwise boost::none.

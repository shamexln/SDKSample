Detailed Description

template<class T_Entity>
class DFL::Mdib::NextLocalMdibViewIterator< T_Entity >

Iterator type to iterate over an MDIB and a change set for given MDIB change type.

The instances are typically created by the NextLocalMdibViewRange type that provides the begin and end iterator.

Factory for end() iterator.

The only internal end() iterator that is always valid is from the MDIB. Depending on the MDIB change type insert and update ranges are not allowed to be accessed.

The dereference and pointer member require to have an lvalue to return. Thus it is not possible to have only a shared_ptr that might be empty. For invalid iterators (end) no entity exists. To represent that value a boost optional is used.

Detailed Description

template<class T, unsigned short replFactor = 1>
class HSM::InPort< T, replFactor >

Input ports, delivering events to the owning state machine.

The InPort class has a "FAN IN" which is limited by template parameter REPL_FACTOR - this means that many outports may be connected to one single InPort. The owner of the inport can ask the event through which port-index it was received.

Template Parameters
TInterface definition class for events received through this port
replFactorMax number of ports that can be connected to this port

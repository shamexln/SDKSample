Detailed Description

template<class T_ParentEntity, class T_EntityAccess>
class DFL::Mdib::Impl::MdibChildIterator< T_ParentEntity, T_EntityAccess >

An iterator that holds an ordered range of handles to child entities modeled as variants.

This iterator can be used with any object that can access entities with the following method signature:

template<class T_Entity>
SharedPtr<T_Entity> entity(const BasicHandle<T_Entity>& handle) const;

Contract: returns pointer to the entity with given handle if existing or nullptr otherwise.

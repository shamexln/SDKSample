Detailed Description

Reperesents a date-day.

The range of valid values is from 1 inclusive to 31 inclusive. But it can hold invalid values which are outside of this range. This class is typesafe wrapper around an integral type. Arithmetic and relational operators are defined to make it behave like an integral type.

Detailed Description

Interface to create MdibXPath expressions.

MdibXPath can be created from an enumeration or plain DFL::Locale::Utf8.

This class also provides an interface to retrieve the root or extension of the identification from a location/patient context.

◆ predefinedExpression()

boost::optional<MdibXPathExpression> S31::Safety::MdibXPath::predefinedExpression ( ) const

Returns the predefined (enum) expression if available.

Since multiple XPath expressions can lead to the same element, internally a normalization might be required.

Example: /pm:State[@DescriptorHandle='foo'][@ContextAssociation='Assoc']/pm:Identification[1]/@Root points to the same value as /pm:State[@ContextAssociation='Assoc'][@DescriptorHandle='foo']/pm:Identification[1]/@Root.

Returns
The predefined MdibXPathExpression if resolvable, otherwise boost::none.

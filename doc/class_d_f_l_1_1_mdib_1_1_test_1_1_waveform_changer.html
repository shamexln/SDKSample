Detailed Description

A recurring task that changes waves in the given Mdib.

Clocks

The repeating notification is controlled by the SteadyClock while Mdib timepoints use the SystemClock. SystemClock changes have no effect on notification interval. The usage of both clocks is synchronized from time to time (see m_syncSystemAndSteadyClockCounter).

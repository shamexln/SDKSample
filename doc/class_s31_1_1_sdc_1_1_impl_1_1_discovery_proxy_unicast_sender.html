Detailed Description

Send WS-Discovery one-way messages as Http requests to a Discovery Proxy.

This class creates an S31::Sdc::Impl::OutgoingUnicastMessageJob to transport a WS-Discovery one-way message (hello or bye) provided to its S31::Sdc::Impl::IOutgoingSender interface. This class also provides retry scheduling, if a message fails to be delivered with a non-fatal error.

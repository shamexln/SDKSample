static Decimal DFL::Mdib::DecimalMath::normalizeDecimal ( const Decimal &  number)
staticnoexcept

Normalizes value and exponent.

Trailing zeros of the value are removed by incrementing the exponent. E.g.,

  • 34.30 will be converted to 34.3
  • Decimal(3430, -2) -> Decimal(343, -1)
  • Decimal(100, 0) -> Decimal(1, 2)

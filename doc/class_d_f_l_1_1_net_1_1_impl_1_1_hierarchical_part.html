Detailed Description

template<class FwdIter>
class DFL::Net::Impl::HierarchicalPart< FwdIter >

Container for iterator ranges of an URIs hierarchical part.

These ranges are filled during parsing in DFL::Net::Uri. They provide lightweight access to the respective part of the URI via begin() and end().

template<class FwdIter >
void DFL::Net::Impl::HierarchicalPart< FwdIter >::update ( )
inline

Replaces all boost::none objects with empty ranges at the sensible position in the URI.

E.g. the URI "http://localhost/" has no user info component. During parsing in DFL::Net::Uri, the user info range will be filled with boost::none. Update() is then invoked to have both begin() and end() of the user info range point to the "l" in "localhost" as this is where the user info component would be.

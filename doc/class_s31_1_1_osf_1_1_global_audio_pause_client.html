Detailed Description

Client interface that is intended to send audio-pause activate requests to all connected devices.

Listens to new devices and maintains a list of online/offline devices. Any call to GlobalAudioPauseClient::activateRemoteAudioPause() invokes an audio-pause activation at all connected, online remote devices.

Detailed Description

A uniformly-distributed random bit generator (URBG) that uses the random source injected via DFL::Crypto::Registry::randSource.

  • Implements the concept UniformRandomBitGenerator (see https://en.cppreference.com/w/cpp/named_req/UniformRandomBitGenerator).
  • By default DFL::Crypto::OpenSslUrng is used as random source.

Cryptographically secure usage:

::DFL::Crypto::RandomDevice roll{};
std::uniform_int_distribution<int> dice{1, 6};
std::vector<int> rolls(10, 0);
std::generate(rolls.begin(), rolls.end(), [&]() mutable {
return dice(roll);
});
ASSERT_TRUE(std::all_of(rolls.cbegin(), rolls.cend(), [](int a) {
return a >= 1 && a <= 6;
}));

std::vector<int> rolls(10, 0);
std::generate(rolls.begin(), rolls.end(), [&]() mutable {
return dice(roll);
});
ASSERT_TRUE(std::all_of(rolls.cbegin(), rolls.cend(), [](int a) {
return a >= 1 && a <= 6;
}));

::DFL::Crypto::Registry::randSource(returnOne);

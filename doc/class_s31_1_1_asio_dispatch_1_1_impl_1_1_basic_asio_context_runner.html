Detailed Description

Basic scheduling on boost asio io context.

'Basic' means here especially that execution can not be prioritized. This is suiting for certain tasks e.g. SSL handshakes, or periodic background tasks.

When this class is shut down, it will attempt to finish any pending (ready) work items.

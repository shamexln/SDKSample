Detailed Description

Puts received messages into the "DW-Detailed" Windows Event Logbook.

Messages are formatted according to IN9880.

Precondition
Ensure that the according logbook is installed on your windows machine. Otherwise messages appear in the Application Logbook. For installation execute S32/Framework/Log/Tools/WinEvtLog/InstallDwLogs.ps1 with admin rights.

Detailed Description

Observer to client subscription(s).

This observer is intended to be invoked if the subscriptions of an SDC device change. This is either there a new subscribers and/or subscribers are gone.

Callback that is invoked when a subscription is lost unexpectedly.

If invoked, then the identified subscription has failed due to delivery timeout or error. To monitor for reconnect, the observer may wait for subsequent clientSubscriptionsChanged() callback invocation, and read information about the current active subscriptions from the observed subject.

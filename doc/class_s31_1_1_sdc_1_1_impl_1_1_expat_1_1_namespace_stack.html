Detailed Description

Stack container for xml namespace to prefix mapping.

This is intended for use by the expat parser to add namespaces as they are discovered, and remove them as they go out of scope.

Returns the local part of a qualified name for given namespace.

The method compares the prefix portion of the qualified name with the given namespace. If these are found to match in the namespace stack, then the local part of the name is returned. Otherwise nullptr is returned.

Detailed Description

template<typename BasicSem>
class DFL::Thread::SemGuard< BasicSem >

Automatically acquires and releases a semaphore.

The constructor acquires ownership of semaphore and acquires it. The destructor releases the semaphore. If guard is constructed with adaptLock it is assumed that the given semaphore is already acquired. Then acquire is not called in the constructor but the destructor still releases the semaphore.

Template Parameters
BasicSemThis concept is fulfilled if type has a acquire() and release() member function.

Public Member Functions

Implementation of modify functions.

Derived classes have to implement two ways to perform modification. MdibAccess will call exactly one of this for a change set. Note that a modifier must not make changes that would alter the MdibChangeType of a write operation, since multiple reports cannot be made for the same MdibVersion.

Detailed Description

template<std::size_t memSize, class TYPE>
class HSM::AlignSizeStatic< memSize, TYPE >

Computes a compile time constant which is equal to the parameter memSize but aligned to the parameter TYPE. Performance tuned by not using multiplication or division operation. E.g. memSize = 7, TYPE = UINT32 —> SIZE = 8 ( 2* sizeof(UINT32) ) 0000 0111 + 0000 0011 = 0000 1010

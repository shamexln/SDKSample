Detailed Description

template<std::size_t Size>
class S31::Log::ShortText< Size >

Holds a short text.

Intended to be used as a return type of functions that need to be noexcept and therefore cannot use std::string. The text cannot be modified.

Instead of mimicking the interface of a string or a string_view this just provides access to a string_view. So it's called ShortText and not ShortString.

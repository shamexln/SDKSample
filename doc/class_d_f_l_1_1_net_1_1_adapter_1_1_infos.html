Detailed Description

Holds all information about every network adapter known to the OS.

The information can be retrieved via get.

std::cout << "\nAll network interfaces:\n";
const auto& infos = ::DFL::Net::Adapter::Infos::get();
std::cout << infos << "\n\n";
DFL::Net::Adapter::Infos::get
static Infos get()
Gathers network adapter information from OS.

Access to each network adapter information is given via an STL compatible container API.

std::cout << "\nNetwork interfaces one by one:\n";
for (auto&& i : infos)
{
std::cout << i << '\n';
}

Since the name and index of an info object are unique this container provides according find-methods:

Detailed Description

An alert condition contains the information about a potentially or actually hazardous situation.

Examples: a physiological alarm limit has been exceeded or a sensor has been unplugged.

List of handle references to sources (e.g., metrics) that cause the alert condition.

Examples:

  • if a source is present: the heart rate metric is the source for a "heart rate high" alert condition.
  • if no source is present: a "cable disconnection" alert condition has no specific source.

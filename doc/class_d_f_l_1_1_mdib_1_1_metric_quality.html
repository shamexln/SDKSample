◆ validity()

MeasurementValidity DFL::Mdib::MetricQuality::validity ( ) const
inline

Defines the validity of a metric as defined in MeasurementValidity.

While the validity is MeasurementValidity::MeasurementOngoing or MeasurementValidity::NotAvailable, the enclosing metric value shall not possess a determined value.

Note
In case of other values the enclosing metric is allowed to possess a determined value.

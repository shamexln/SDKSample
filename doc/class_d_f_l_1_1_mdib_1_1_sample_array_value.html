Detailed Description

Type that contains sequences of values, i.e., sample arrays.

The MetricQuality element relates to all samples.

Timestamp (see base: AbstractMetricValue) refers to the first value of the array. The individual timestamps of the values can thus be computed from the sample rate (see RealTimeSampleArrayMetricDescriptor::samplePeriod).

If MetricQuality cannot be applied to all samples due to, e.g., some invalid values, it is allowed to set MetricQuality::validity to MeasurementValidity::Questionable or MeasurementValidity::Invalid for the whole sample array.

Note the semantics of annotations as described in Differences to IEEE model.

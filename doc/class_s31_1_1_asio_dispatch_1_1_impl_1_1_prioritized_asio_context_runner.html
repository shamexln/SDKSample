Detailed Description

Prioritized scheduling on boost asio io context.

The Executor provided by this class should be used to construct boost::asio objects (i.e. sockets, timers, etc). The work initiated by these objects will be added to a priority queue for execution during AsioContextRunner::runFor(). When this class is shut down, it will attempt to finish any pending (ready) work items. These items are expected to to be owned by a different io_context, so need to be released from here.

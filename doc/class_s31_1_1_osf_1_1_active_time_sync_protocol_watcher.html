Detailed Description

Monitor active time sync protocol of connected devices.

This utility class provides support for detecting that the active time sync protocol of a connected device is not MDC_TIME_SYNC_NTPV3 or MDC_TIME_SYNC_NTPV4.

Notification that a connected device's active time sync protocol is not MDC_TIME_SYNC_NTPV3 or MDC_TIME_SYNC_NTPV4.

Caller provides S31::Sdc::EndpointReference that identifies the connected device, the optional DFL::Mdib::CodedValue that identifies the activeSyncProtocol, (or none if activeSyncProtocol is not set, or the clock is not unique), and the DFL::Mdib::Mdib from the remote device.

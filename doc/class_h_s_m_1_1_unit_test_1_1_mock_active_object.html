Detailed Description

Mock class to be used in unit tests of HSM state machines. Serves as run time environment for the state machines and provides a mock timer implementation for use in the tests. The timer implementation holds a pseudo time which itself never moves on, only when told by the the method advanceTime. Any timers used by the state machine under test can be queried whether they are running and for their next timeout and can be either fired directly or indirectly by advancing the pseudo-time. Also any tick timers can be fired directly or by indirectly by calling the tick method. Uses MockDebugLog to allow accessing the log output during unittesting

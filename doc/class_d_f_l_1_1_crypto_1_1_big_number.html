Detailed Description

Represents an arbitrary big integer number.

It wraps an OpenSSL BIGNUM handle which can be constructed from a container of bytes.

Constructor & Destructor Documentation

◆ BigNumber()

DFL::Crypto::BigNumber::BigNumber ( )

Construct an empty big number.

Empty meaning that it isn't a number at all. Not even 0. It is just an empty shell. It does not allocate any resources and thus is cheap. It can be used as a placeholder to be assigned later, e. g. initializing a container of big numbers.

const auto a{BigNumber{}};
EXPECT_EQ(a.bignum(), nullptr);
EXPECT_NE(a, BigNumber::fromBytes(Bytes{0x00}));

Constructs a big number from a container of bytes.

const Bytes m_123{0x7b};
const Bytes m_00123{0x00, 0x00, 0x7b};
const Bytes m_456{0x01, 0xc8};
EXPECT_EQ(BigNumber{}, BigNumber::fromBytes(Bytes{}));
EXPECT_TRUE(BigNumber::fromBytes(m_123).bignum());
// leading zeros are skipped
EXPECT_EQ(BigNumber::fromBytes(m_123), BigNumber::fromBytes(m_00123));

◆ release()

MutableType DFL::Crypto::BigNumber::release ( )
noexcept

Releases the wrapped OpenSSL BIGNUM handle.

The caller is now responsible for freeing the handle.

auto bn{BigNumber::fromBytes(m_123)};
const auto* handle{bn.bignum()};
auto* releasedHandle{bn.release()};
/*
* CAUTION:
* You are now responsible for freeing the released handle.
*/
const auto freeHandle{
ScopeExit{[releasedHandle]() { ::BN_free(releasedHandle); }}};
EXPECT_EQ(handle, releasedHandle);
EXPECT_EQ(bn, BigNumber{});

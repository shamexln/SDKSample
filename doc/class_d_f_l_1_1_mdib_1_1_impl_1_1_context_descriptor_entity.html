Detailed Description

template<class T_Entity, class T_DescrBuilder, class T_StateBuilder, class T_Parent>
class DFL::Mdib::Impl::ContextDescriptorEntity< T_Entity, T_DescrBuilder, T_StateBuilder, T_Parent >

Context descriptor MDIB entity.

A context descriptor MDIB entity is different to an MdibEntity in the way that it possesses no state. States of the context descriptor entity are stored in special containers (see ContextStateEntity) and possess their own handle.

Detailed Description

Overwrites several configurations/behaviours of this library.

This is specifically useful in untit-tests but in sample applications too. All changed configurations are only in effect as long as an instance of this class exists. Upon destruction all modifications are undone.

DFL::Password::Test::OverwriteEnv::OverwriteEnv ( )

Overwrites configurations/behaviours of this library.

  • Persistence is exchanged with a fresh volatile one.
  • The secret used for hashing password is set to a hardcoded one.

Parameters

lineThe line number in source file, where the exception occurred.
Returns
Nothing because an exception is always thrown in the function body

This function must not be called directly, it will only be used by THROW_ macro. It will never return; the exception will be built internally and thrown from inside this method.

Use throw BaseException(message, file, line) instead!

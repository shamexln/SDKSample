Detailed Description

Interface for SOAP web services.

Interface must be implemented by all web services that the DPWS stack should handle. Typically this interface should be generated from gSOAP in the <name>Service.h files.

Currently (2.8.8) gSOAP does not include the interface in the class definition. That is why this interface must be implemented by the application developer.

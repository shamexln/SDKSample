Detailed Description

template<class Container>
class DFL::Utils::ConstContainer< Container >

Constant container which meets the C++ named requirement Container.

This class shall support customer in writing container-like types. In contrast to the STL containers, it is designed to be used as base class:

using ConstContainer = ::DFL::Utils::ConstContainer<std::vector<int>>;
class GoodContainer : public ConstContainer
{
public:
using ConstContainer::ConstContainer;
};

Exchanges the contents of the container with those of other.

using Underlying = std::vector<std::string>;

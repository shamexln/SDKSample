Detailed Description

Container for Uniform Resource Identifier (URI).

The class acts as a lightweight wrapper for a URI string and provides access to an URI's individual parts. See RFC3986 for reference. The string provided is parsed on construction and throws a runtime error if the string is not a valid URI. Relative URIs are treated as invalid.

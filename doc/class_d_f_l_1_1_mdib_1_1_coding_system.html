Coding system version

Version of the system - if relevant.

Note that a well-maintained coding system does not need the version reported, because the meaning of codes is consistent across versions. However this cannot consistently be assured. and when the meaning is not guaranteed to be consistent, the version should be set.

Detailed Description

Represents a coding system with codes to identify concepts.

An example coding system is the ISO/IEEE 11073-10101 nomenclature to identify metrics such as SPO2.

Detailed Description

Get the texts that match the filter in a store.

This implements the lookup needed for the SDC GetLocalizedText() request.

Additional/dissenting lookup rules to the matching rules described in the standard

The content with respect to 'text width' and 'number of lines' of a LocalizedTextFilter is matched as follows:

  • if LocalizedTextFilter::textWidths() and LocalizedTextFilter::numberOfLines() contain at most 1 element each, then all text elements match that match both of these filter elements (all that is less or equal)
  • if LocalizedTextFilter::textWidths() or LocalizedTextFilter::numberOfLines() contain more than 1 element, then all text elements match
  • With respect to 'text width' only: a text without textWidth matches always.

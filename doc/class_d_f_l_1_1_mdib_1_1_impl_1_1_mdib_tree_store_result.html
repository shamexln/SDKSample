Detailed Description

template<typename T_Entity>
class DFL::Mdib::Impl::MdibTreeStoreResult< T_Entity >

Definition of a result object that ensues from any MDIB tree store operation.

The MdibTreeStoreResult encloses

  • an operation result status and
  • depending on the operation a map of affected MDIB entities (optional).

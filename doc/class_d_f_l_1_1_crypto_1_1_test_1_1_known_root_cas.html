Detailed Description

Provides access to known root cas.

These root cas are typically used in the context of tests or samples.

Certificate DFL::Crypto::Test::KnownRootCas::draegerTest ( )
inlinestatic

This is the test root ca for all draeger SDC devices.

This root allows devices to accept certificates which are not signed by the signing authority. Thus the process of creating signed certificates is reduced. On the other hand the private key of this root is known. By no means should this root ever be used in production.

Detailed Description

template<class T_Entity, class T_Mdib>
class DFL::Mdib::Impl::MdibIteratorFromHandles< T_Entity, T_Mdib >

An Iterator that holds a range of handles in a multimap and goes over the entities in the also given Mdib.

If the iterator meets a handle for which it cannot find an entity the iterator stops an this handle (goes to end() immediately)

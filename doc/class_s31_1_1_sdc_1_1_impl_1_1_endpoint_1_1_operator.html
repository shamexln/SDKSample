Detailed Description

template<class T_Result>
class S31::Sdc::Impl::Endpoint::Operator< T_Result >

Manages reception of job responses and retries to send requests on failure.

The endpoint operator receives results from outgoing blocking HTTP requests and retries sending until successful delivery or until no more URLs to test against are available.

Detailed Description

template<class InType, unsigned short replFactor, class OutType = NoInterface>
class HSM::ReceiverPort< InType, replFactor, OutType >

A receiver port represents a communication endpoint.

Template Parameters
InTypeInterface definition class for the events received through this port
replFactorMax number of ports that can be connected to this port
OutTypeInterface definition class sent through this port, can be NoInterface, if this is only a receiver

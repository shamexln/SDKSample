Detailed Description

Implementation of S31::Sdc::Impl::HttpServerHandler interface for SDC functionality.

Http GET requests (for the WSDL file) are handled directly. Http POST messages are dispatched and the requests are forwarded to specific implementations with help of the S31::Sdc::Impl::LocalDeviceRegistry. Soap handles are obtained from the provided S31::Sdc::Impl::ISoapContextPool.

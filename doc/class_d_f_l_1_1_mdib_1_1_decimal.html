Detailed Description

Represent a numeric value with fixed point.

The representation of this class uses an integer value with an additional exponent (base 10). It is designed to preserve the precision from the XML schema decimal datatype and also covers numeric metric values from the S14 platform without conversions.

For conversion to/from double and std::string Model/DecimalIo.h provides corresponding functions.

The class is explicitly defined without

  • not a number
  • overflow
  • underflow

support. The reasons are

  • XML Schema decimal requires to have a valid numeric value
  • S14 also does not support invalid states of its equivalent data types.

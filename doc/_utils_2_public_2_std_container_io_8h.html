Detailed Description

IO streaming operators for std containers.

Note
The CPP standard says that it is undefined behavior to add declarations or definitions to namespace std (or to any namespace nested within std). StdContainerIo.h should not cause any undefined behavior as it only adds missing streaming operators to the namespace. Further information: https://wiki.sei.cmu.edu/confluence/display/cplusplus/DCL58-CPP.+Do+not+modify+the+standard+namespaces

Usage:

namespace
{
const int InputArray[] = {0, 1, 10, 100};
const int SizeOfInputArray = sizeof(InputArray) / sizeof(*InputArray);
}
TEST(StdContainerIo, testPair)
{
std::ostringstream str;
std::pair<int, std::string> const p = std::make_pair(1, "one");
str << p;
ASSERT_EQ(std::string("{1,one}"), str.str());
}
TEST(StdContainerIo, testVector)
{
std::ostringstream strm;
std::vector<int> const empty;
strm << empty;
ASSERT_EQ(std::string("[]"), strm.str());
strm.str(std::string());
std::vector<int> const v(InputArray, InputArray + SizeOfInputArray);
strm << v;
ASSERT_EQ(std::string("[0,1,10,100]"), strm.str());
}

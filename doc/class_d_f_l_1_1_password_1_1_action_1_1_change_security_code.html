bool DFL::Password::Action::ChangeSecurityCode::checkQuality ( const Store::RawSecurityCode &  secCode) const

True if given securitycode conforms the quality rules otherwise false.

A securitycode is conformant if it ...

  • ... is 8 characters long
  • ... only contains digits
  • ... consists of more than one digit
  • ... has neither ascending/descending digits
  • ... is not the current securitycode

const std::string containsNewLine{"01233\n21"};

{"21082010"}};

Detailed Description

Contains detailed information about a location in contrast to identifiers for identification purposes.

The definition of this class is derived from the HL7 standard for patient locations. It contains the six fields to specify the location of the patient within the hospital (facility).

They are listed in IHE PCD TF volume 2 Appendix C7. From the 6-tuple within Dräger only Facility (hospital), PointOfCare (care unit), and Bed are typically used. All fields are encoded in UTF-8 for unicode support.

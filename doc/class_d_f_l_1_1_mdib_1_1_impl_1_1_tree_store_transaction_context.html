Detailed Description

Set of data that is used as a transaction context on insert and remove in MdibTreeStoreBuilder.

Member Typedef Documentation

◆ InitFlags

using DFL::Mdib::Impl::TreeStoreTransactionContext::InitFlags = ManyInitFlags<InitFlag>

Indicates if a certain map hosted by the transaction has been accessed once.

During a transaction, entity maps are added to the transaction when needed. To avoid unnecessary copies in case a map is accessed multiple times, the InitFlags class stores a flag that indicates if a typed map has already been initialized within the transaction context.

Detailed Description

ZonedTime represents a logical paring of a time zone and a system clock time point.

The local time is obtained by applying the according time zone offset to the system clock time point.

using namespace ::DFL::Chrono;
const auto now{SystemClock::now()};
{
std::cout << "current zone:\n";
const auto a{ZonedTime{now, currentZone()}};
std::cout << "zoned time: " << a << '\n';
std::cout << "system time: " << Convert::toIso8601(a.systemTime())
<< '\n';
std::cout << "local time: " << a.localTime() << '\n';
ASSERT_EQ(a.systemTime(), now);
}
{
std::cout << "custom zone:\n";
const TimeZone tz{IanaFormat{"XYZ+02:00"}};
const auto a{ZonedTime{now, tz}};
std::cout << "zoned time: " << a << '\n';
std::cout << "system time: " << Convert::toIso8601(a.systemTime())
<< '\n';
std::cout << "local time: " << a.localTime() << '\n';
ASSERT_EQ(a.systemTime(), now);
}

◆ ZonedTime() [1/2]

DFL::Chrono::ZonedTime::ZonedTime ( const SystemClock::TimePoint &  sysTime)
explicit

Takes a system clock time point and thus holds a utc zoned time.

Thus no offset to the system time is applied when a local time is calculated.

{
const ZonedTime zt(sysTime_a);
ASSERT_EQ(sysTime_a, zt.systemTime());
ASSERT_EQ(sysTime_a.time_since_epoch(), zt.localTime().time_since_epoch());
}

Takes a system clock time point and a time zone.

The given time zone is used for converting system time into local time.

{
const ZonedTime zt(sysTime_a, tz);

}
{

ASSERT_EQ(sysTime_b, zt.systemTime());
ASSERT_EQ(locTime_b, zt.localTime());
}

static void DFL::Net::Impl::NetworkMonitorRegistry::notifyNetworkChange ( )
static

Call this function if you know there are network configuration changes.

In some situations the operating system is not aware of network changes, e.g., when having some network switch in front of the network adapter. To provide change events this function needs to be called.

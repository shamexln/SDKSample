Detailed Description

template<class T_Entity>
class DFL::Mdib::NextLocalMdibViewRange< T_Entity >

Range type for a specific MDIB entity type for an MDIB with an applied MDIB change set according to the change type.

Use this in case of getting all entities after applying an MDIB change set. Typical use case is a check before applying a change set.

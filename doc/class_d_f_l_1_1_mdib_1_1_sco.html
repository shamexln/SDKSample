Detailed Description

template<class T_Entity, class T_Parent>
class DFL::Mdib::Sco< T_Entity, T_Parent >

A service control object (SCO) to define remote control operations.

Any AbstractOperationDescriptor::operationTarget within an SCO shall only reference sibling or child descriptors of the SCO enclosing entity.

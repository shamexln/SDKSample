Registers the callback for time/clock configuration.

It registers the given callback in the SetService for the set time zone and set NTP server operation and enables the operation (DFL::Mdib::OperatingMode::Enabled). The operation has to be in the MDIB with correct type and target operation set to ClockDescriptor.

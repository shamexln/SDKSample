◆ bicepsXmlReportMessage()

const std::string& S31::History::Impl::Bundle::bicepsXmlReportMessage ( ) const

Concatenation of all MDIB reports (raw XML or gzip compressed).

It include the HistoricReport xml tag so that it can be used afterwards without further analysis. As long as additional reports might be added the XML data is uncompressed. Afterwards it will be gzip compressed with sync end to allow concatenating it with other gzip compressed chunks.

◆ mdib()

const std::string& S31::History::Impl::Bundle::mdib ( ) const

Optional MDIB as XML string.

To save memory not for all MDIB report bundles an MDIB snapshot is stored. From this it follows that this field might be empty if no such MDIB snapshot is available. To find a sync point previous data entries must be evaluated.

Current number of MDIB reports in this instance.

This data structure collects MDIB reports and store in this variable the current number. It could be used to check if the bucket is full based on current configuration.

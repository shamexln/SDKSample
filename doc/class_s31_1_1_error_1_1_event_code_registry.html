Detailed Description

Provides access to module events at runtime.

This static/Singleton class handles events (errors and states) that are notified to the product.

void S31::Error::EventCodeRegistry::addEvent ( EventCode  id)

Adds the error state to the set of active error state if there is not a state with this id already.

If the the error state is active already then nothing is done. The listener is notified only if the error state was not active already.

Detailed Description

Visitor class to call a function for each entity's localized texts.

With the EntityPart it's possible to define which localized texts locations in the DFL::Mdib::Mdib or DFL::Mdib::MdibChangeSet are visited.

Note that in the case of a change set the removed entities are not visited.

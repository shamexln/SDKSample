Detailed Description

A field based time point with a resolution of one month.

This type combines year and month value of a date. Especially the arithmetic operators take care of any year overflow if it occurs.

{
const YearMonth ym = Year(2014) / jan;
ASSERT_EQ((ym.year() - Years(1)) / dec, ym - Months(1));
ASSERT_EQ((ym.year() - Years(1)) / jan, ym - Years(1));
}
{
const YearMonth ym = Year(2014) / dec;
ASSERT_EQ((ym.year() + Years(1)) / jan, ym + Months(1));
ASSERT_EQ((ym.year() + Years(1)) / jan, Months(1) + ym);
ASSERT_EQ((ym.year() + Years(1)) / dec, ym + Years(1));
ASSERT_EQ((ym.year() + Years(1)) / dec, Years(1) + ym);
}

Any arbitrary year-month-dates can be created even some which do not exist. Use YearMonth::ok() to check if the date is valid.

{
const YearMonth ym(Year(2010), aug);
ASSERT_TRUE(ym.ok());
ASSERT_EQ(Year(2010), ym.year());
ASSERT_EQ(aug, ym.month());
}
{
const YearMonth ym(Year(2010), Month(13));
ASSERT_FALSE(ym.ok());
}
{
const YearMonth ym(Year(2010), Month(0));
ASSERT_FALSE(ym.ok());
}

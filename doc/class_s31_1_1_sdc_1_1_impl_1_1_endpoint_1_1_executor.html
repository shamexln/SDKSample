Detailed Description

Provides functionality to repeat sending of outgoing blocking HTTP requests to several URLs.

The endpoint is an abstraction of a receiver of network requests, in particular HTTP requests. It can be used, e.g., for any DPWS message transmission to

  • devices (hosting services),
  • hosted services,
  • subscription managers,
  • notify-to endpoints, and
  • subscription-end endpoints.

The endpoint accepts a job creator function and a callback handler that is used to forward responses to. Internally, the endpoint loops through a prioritized list of URLs, enqueues a job accordingly and invokes the given callback handler once one of the jobs succeeds or the last existing job fails.

Detailed Description

Tests if given device metadata fulfills SDC Glue requirement R0034.

R0034: An SDC SERVICE PROVIDER SHALL implement at least the following BICEPS SERVICEs as one MDPWS HOSTED SERVICE if the SDC SERVICE PROVIDER exhibits the capabilities that require the BICEPS SERVICE:

  • Description Event Service
  • State Event Service
  • Context Service
  • Waveform Service

Detailed Description

template<class T_Entity, class T_Container = RandomEntityAccess<T_Entity>>
class DFL::Mdib::Impl::MdibIterator< T_Entity, T_Container >

A template for iterators for Mdib entity containers.

Used to give Mdib users access to entities without revealing internal container structures.

For now this wraps a std::map iterator, but if needed there could be specializations of this this template of other types that work similar transparent for the end user.

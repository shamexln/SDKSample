Detailed Description

Used for special situations that when the time that an inactivation state ends can no longer be determined from the @DeterminationTime and @InvocationEffectiveTimeout of the belonging pm:AlertConditionState and pm:ActivateOperationState.

An SDC ALERT PROVIDER shall set it to the latest time that an alert signal inactivation state will end automatically.

An example it the audio pause function with a count down when the inactivation will end automatically.

Note
An SDC ALERT PROVIDER shall set @ext:MustUnderstand = "true".

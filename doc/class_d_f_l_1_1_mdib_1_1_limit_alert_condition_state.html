◆ monitoredAlertLimits()

AlertConditionMonitoredLimits DFL::Mdib::LimitAlertConditionState::monitoredAlertLimits ( ) const
inline

Monitored alert limits of the limit alert condition.

Note if the AlertConditionState::activationState() attribute is not set to AlertActivation::On, the presence of the limit alert condition is not detected. From a consumer perspective this is equivalent to the case when the AlertConditionState::activationState() is AlertActivation::On and LimitAlertConditionState::monitoredAlertLimits() is AlertConditionMonitoredLimits::None. The difference is that in the latter case the source for the alert condition is still supervised, but the presence flag is not generated even if the limits are violated.

Detailed Description

Pool for idle outgoing HTTP keep-alive connections.

This pool stores outgoing HTTP keep-alive sockets in SoapPtr contexts. This is the way gSOAP uses keep-alive. One soap context is used for all HTTP requests in a keep-alive connection.

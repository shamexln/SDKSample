Detailed Description

Boost.Asio based network monitor.

The design of this class is derived from the boost::asio::steady_timer. It provides a synchronous wait (waitFor()) and an asynchronous wait function (asyncWait()). Instead of waiting for an elapsing timer, the network monitor wait for network configuration changes (IP address change, ...).

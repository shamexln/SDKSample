Detailed Description

Mocks SystemClock::now() by returning a frozen SystemClock::TimePoint.

By creating an instance of this class, the time point returned by SystemClock::now() will be frozen. The returned time point can be manipulated by the methods setClock and advanceClock. After destructing the instance, SystemClock::now() will work as usual again.

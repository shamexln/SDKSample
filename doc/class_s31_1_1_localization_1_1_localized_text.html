Detailed Description

LocalizedText with number of lines.

Rationale: Mdib's DFL::Mdib::LocalizedText follows BICEPS and doesn't models the 'number of lines' property that OTOH is used within Localization Service.

Users can lookup texts from the localized text store restricted to 'number of lines'. With this class the result from this lookup has the actual 'number of lines' handy for the user.

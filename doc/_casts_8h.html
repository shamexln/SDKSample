Detailed Description

Provides useful cast functions.

Some of these cast functions are provided as a replacement for existing c++ cast functions. exact_cast is a replacement for dynamic_cast. dynamic_cast is able to do downcasts aswell as crosscasts. It does so by using the typeid and traversing the inheritance structure. In cases where the exact target type is known a more efficient implementation can be used. This is provided by the exact_cast. It performs one typeid comparison and a static_cast. Given this class hierachy:

class Base
{
public:
virtual ~Base(){}
};
class Sub_1 : public Base
{};
class Sub_1_1 : public Sub_1
{};

An exact_cast can perform the following conversions.

Base* baseToSub_1 = new Sub_1();
Sub_1* sub_1 = exact_cast<Sub_1*>(baseToSub_1);
Base* baseToSub_1_1 = new Sub_1_1();
Sub_1_1* sub_1_1 = exact_cast<Sub_1_1*>(baseToSub_1_1);

This conversion does not work and throws an exception:

Base* baseToSub_1_1 = new Sub_1_1();
exact_cast<Sub_1*>(baseToSub_1_1); // throws

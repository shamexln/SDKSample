Detailed Description

Variant type to contain MdsAlertSignal or VmdAlertSignal data.

VmdAlertCondition might have signal in the MdsAlertSystem as well as in the VmdAlertSystem. Thus, the is no specific signal type to provide when searching for alert signals belonging to a VMD alert condition.

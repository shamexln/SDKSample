Detailed Description

Volatile part of battery information.

The current power source is designated by ComponentActivation:

  • If BatteryState::activationState equals ComponentActivation::On, the device is running on battery
  • If BatteryState::activationState equals ComponentActivation::Off, the device is in mains operation and currently not able to be run on battery
  • If BatteryState::activationState equals ComponentActivation::StandBy, the device is in mains operation and can be switched to run on battery
  • If BatteryState::activationState equals ComponentActivation::Failure, the battery has a malfunction. Detailed error information should be communicated by using an alert system.
Note
Enumerations ComponentActivation::Shutdown and ComponentActivation::NotReady are undefined for BatteryState.

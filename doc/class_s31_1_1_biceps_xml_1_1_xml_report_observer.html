Detailed Description

Observer interface similar to SubdividedMdibObserver with BICEPS xml changes report.

The SubdividedMdibObserver provides the MDIB changes with C++ objects only, while this observer interface adds the serialized form of the MDIB change set as a BICEPS xml report message.

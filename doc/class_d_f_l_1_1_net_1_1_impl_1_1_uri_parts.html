Detailed Description

template<class FwdIter>
class DFL::Net::Impl::UriParts< FwdIter >

Container for iterator ranges of an URI.

These ranges are filled during parsing in DFL::Net::Uri. They provide lightweight access to the respective part of the URI via begin() and end().

template<class FwdIter >
void DFL::Net::Impl::UriParts< FwdIter >::update ( )
inline

Replaces all boost::none objects with empty ranges at the sensible position in the URI.

E.g. the URI "http://localhost/path?foo" has no fragment component. During parsing in DFL::Net::Uri, the fragment range will be filled with boost::none. Update() is then invoked to have both begin() and end() of the fragment range point to the URIs end.

Detailed Description

Base class for different derived classes which all rely on the same representation in memory, which is defined here. Memory consumption is reduced, by avoiding any virtual methods and using bitfields of appropriate size for the members. The m_dataOrPort member is used as either data or port pointer depending on the life-cycle of the event. When sending events they first contain a data pointer, but when the event is put into the queue of the active object a pointer to the receiving port is put into this member and the data is store adjacent to the header in the queue.

The copy constructor and the assignment operator are not declared private here, because the default implementation with shallow copy is sufficient here and is used.

Detailed Description

Interface to customize various policies for Https connections.

Https connections to different endpoints may require specialized rules or policies. For example, connection to a discovery proxy server does not require Sdc participant role checks, and low or high priority Sdc services require different quality of service configurations. This interface provides a collection methods to customize the behavior of an Https connection.

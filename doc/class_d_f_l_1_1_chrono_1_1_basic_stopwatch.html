Detailed Description

template<class ClockType, class DurationType = typename ClockType::duration>
class DFL::Chrono::BasicStopwatch< ClockType, DurationType >

Calculates the elapsed duration between start and stop.

This type implements the common behaviour shared between all stopwatch types. It can be customized on a clocktype for specifying the time source. For convenience aliases like DFL::Chrono::Stopwatch are provided.

The common behaviour is as follows:

To start the stopwatch you can either call start on a default constructed stopwatch ...

Stopwatch w{};
w.start();
EXPECT_TRUE(w.isRunning());

StopwatchD< SteadyClock::duration > Stopwatch

... or use autoStart on construction for starting it immediately.

const Stopwatch w{autoStart};
EXPECT_TRUE(w.isRunning());
DFL::Chrono::autoStart
DFL_CHRONO_EXPIMP const AutoStartType autoStart
Stopwatch created with this value will start automatically.

Calling elapsed on a running stopwatch returns a snapshot of the elapsed time.

w.start();
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 1_us);
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 2_us);
DFL::Thread::ThisThread::sleepFor
DFL_THREAD_EXPIMP void sleepFor(const Chrono::Milliseconds &ms)
Blocks the execution of the current thread for the specified duration.

After calling stop the elapsed time does not change anymore.

w.stop();
EXPECT_EQ(w.elapsed(), 2_us);
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 2_us);

reset can be used to set the elapsed time to zero.

w.reset();
EXPECT_EQ(w.elapsed(), 0_us);
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 0_us);

The measurement can be continued by calling start on a stopped stopwatch again.

w.stop();
EXPECT_EQ(w.elapsed(), 2_us);
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 2_us);
w.start();
sleepFor(1_us);
EXPECT_EQ(w.elapsed(), 3_us);

isRunning can be used to retrieve the current state.

Furthermore the following behaviour is implemented:

  • Calling start on a running stopwatch does nothing.
  • Calling stop on a stopped stopwatch does nothing.
  • Calling reset on a running stopwatch is the same as calling:
    • stop
    • reset
    • start
